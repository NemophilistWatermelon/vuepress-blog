# 使用 NRM

  • 安装方式
    npm install -g nrm
  • 查看镜像源
    nrm list
  • 切换镜像源
    nrm use xxx
  • 当前使用镜像源
    nrm current
  • 镜像测速
    nrm test

# mac 注意

  • mac m1 安装方式
